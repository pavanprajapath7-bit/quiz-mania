# Quiz Game 🎯 (Python)

A simple Python CLI quiz game with scoring and high score saving system.

## Features
- Multiple Choice Quiz (MCQ)
- Score System
- High Score stored in a JSON file
- Replay option

## Files
- `main.py` → main quiz logic
- `questions.py` → quiz questions
- `highscore.json` → created automatically after playing

## How to Run
```bash
python main.py
